What Is Software ?

Software Is A Collection Of Instruction That Instruction Tell The Computer How To Work.
In Simple Terms You Open The Calculator App Then Enter The Numbers Like 2 + 2 And Press Equal Then Answer Will Be Showed , How It Is Displayed ? In These Example Telling The Computer To Perform The Addition Operation By Clicking The Addition Button .

Software Is Mainly Divided Into 2 Types
1.System Software
2.Application Software

What Is System Software ?

System Software Is A Software Which Is Used To Create The Interface Between The User And Computer. Some Examples Can Be Operating Systems, Antivirus Softwares, Disk Formatting Softwares, Computer Language Translators Etc.

What Is Application Software ?

Application Software Is Software Which Is Designed To Perform The Specific Operations And Tasks. I Will Give The Example You Want To Do Some Additon, Multipication Then You Probably Use The Calulator App To Complete Your Tasks.In These Example Addition , Multipication Are The Specific Operation To Do Perform.
