What is Web ?

World Wide Web is also known as a Web, it is a collection of websites or web pages stored in web servers and connected to local computers through the internet. These websites contain text pages, digital images, audios, videos, etc. Users can access the webpages or websites from any part of the world over the internet using their devices such as computers, laptops, cell phones, etc.

Website are developed with the some languages like html , css , js .
